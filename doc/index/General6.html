The HMI API provides a synchronous high-level communication with the 3DTouchPad devices.
The 3DTouchPad SDK installation provides an API implementation and a few Windows and Linux applications (App) as source code together with precompiled binaries.
The Windows template for HMI applications is an Visual C++ 2010 project for Win32 executables with the HMI API related project configuration.
