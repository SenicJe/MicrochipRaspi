Windows Template

The Windows template for HMI applications is an Visual C++ 2010 project for Win32 executables with the HMI API related project configuration.  It is located at apps/Windows/template.

It links against mchp_hmi.lib and mchp_hmi.dll located at api/lib in the installation directory.
