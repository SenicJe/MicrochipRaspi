Usage

Please connect a 3DTouchPad device to the machine and terminate all applications using this device before running this demo.

After starting fingerpos.exe you should see the following console-window:

The application streams finger-positions for about 10 seconds and terminates afterward.

During this 10 seconds it repeatedly ouputs three lines with the data from hmi2d_finger_pos_list_t.  This list contains data for up to 10 fingers consisting of an internal ID for the finger which is output in the first row, the horizontal X-position in the range from 0 to 511 in the second row and the vertical Y-position in the range from 0 to 383 in the last row.

Fingerpos disables 3D sensing and 2D features like mouse and keyboard-shortcuts which get restored before the application terminates after the timeout.
