Usage

Please connect a 3DTouchPad to the machine and terminate all applications using this device before running this application.

The application has to be invoked with the path to an 3DTouchPad’s 2D firmware (an .hex file) as the only parameter.

update2d 3DTouchPad.hex

After calling the application this way a it will ask whether the firmware should really be flashed.  After answering with ‘yes’ the application continues to update the firmware:

During the update the application switches the 3DTouchPad into the bootloader mode, unlocks and erases the firmware followed by the programming and finally switching back in the normal operation mode.  During programming the start addresses of the written blocks is reported to the screen.

As the necessary mode-switching is seen by the operating system unplugging and plugging of devices for which it has to provide the drivers this steps might take some while.

During the update process 2D and 3D sensing, mouse and keyboard-shortcuts are not available.
