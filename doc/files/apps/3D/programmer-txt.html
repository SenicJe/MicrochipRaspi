Programmer Application

The programmer application demonstrates how to update the Loader Library and the GestIC Library.

It’s source code for Windows is located at apps\Windows\programmer and the source code for Linux is at apps\Linux\programmer.  There is a Windows executable at apps\Windows\bin.

The files Library.c and Loader.c contain the firmware images in form of a hmi3d_update_image_t-compatible variable declaration.  To create this representation for different firmware files the 3DTouchPad SDK provides the MGCConv-tool.

An introduction into the topic could be found at 3D Firmware Update.

Usage

Please connect an HMI device to the machine and terminate all applications using this device before running this application.

When starting programmer.exe you should see a console window similar to the following:

The programmer is starting with flashing the Library Loader Updater.  Then it waits until the Loader Updater has finished its job and continues with flashing the HMI Library.

During this process it writes the progress to the console window but expects no input.

The programmer disables all 2D features like mouse control and keyboard-shortcuts.  After the firmware update the device has to be un- and replugged in order to be fully operational again.
