Version 1.0.0

  • Updated to the new communication protocol of GestIC Library 1.0
  • Added new function gestic_set_output_enable_mask for setting data-output that replaces gestic_data_stream (deprecated)
  • Added support for Touch Detection
  • Added support for AirWheel
  • Added support for Noise Power
  • Moved instruction-related function to the public API
  • Added support for reading back parameters
  • Added function for reading the GestIC Library Version from the device
  • Simplified and cleanup of streaming and synchronization

Version 0.4.21

  • Added gestic_flick_outin-flag

Version 0.4.19 (compared to API)

  • Added source-code for the GestIC-API
  • New function gestic_set_enabled_gestures to change what gestures could be detected
  • Data related to gesture-detection changed
  • Additional data related to glider position
